This UI design project was a conceptual exercise and not an official product of Big Apple Donuts & Coffee.

The design incorporates the brand's visual identity, including the logo, fonts, color schemes, and menu items, for educational and illustrative purposes only. The project aims to demonstrate an understanding of UI/UX principles and their application in a real-world context.

The work presented here is a UI design for a food ordering system.

Due to the limitations of showcasing interactive prototypes in a static portfolio format, only screenshots of the design are available.

The actual prototype was developed using Justinmind and featured interactive elements such as clickable buttons and dropdown menus.

This is the main menu that the customer will see when they first start using the Food Ordering System.

The main menu provides a clear overview of the three main categories: Donuts, Mini Donuts, and Beverages.

Navigation is facilitated by a home button, a back button, and a "My Order" button for viewing the current order.

Each category displays a visually appealing grid of products that displays the product name and its price.

Out-of-stock items are clearly indicated.

Customers can easily select desired items, specify quantity, and add them to their order.

A "Cancel" button allows users to easily discard individual item selections.

The order summary screen displays a clear list of selected items and quantities.

Customers can easily remove items from the order using the trash icon.

A "Proceed to Payment" button initiates the checkout process (the specific payment flow was not designed in this phase).

A "Cancel Order" button allows users to discard the entire order.

Customers can choose their preferred methods of payment, either by card, online banking, or paying at the counter.
