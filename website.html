An online graphic design tool built with HTML, CSS, and Fabric.js (a JavaScript HTML5 canvas library).

Designed for non-technical users, the tool enables quick mockup creation for items like t-shirts and signboards, bypassing the need for accounts on platforms like Canva.

Users can create custom designs and export them in common image formats (PNG, JPEG) and even in PDF format.

A simple 24 hour analog clock built with HTML, CSS, and JavaScript.

I made this because I can't find any 24 hour analog clock in physical stores, and it doesn't seem like an online one exists either. So, I decided to make my own to fulfill my own needs.
